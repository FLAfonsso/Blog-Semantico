Um blog semântico e responsivo com tema dark, desenvolvido para compartilhar aventuras, viagens e experiências incríveis.

![Blog Preview](https://via.placeholder.com/800x400?text=Blog+Preview)

## ✨ Funcionalidades

- 🎨 **Tema Dark** - Design moderno e agradável para leitura noturna
- 📱 **Responsivo** - Adapta-se perfeitamente a dispositivos móveis, tablets e desktop
- 📝 **Páginas completas**:
  - Página inicial com lista de posts
  - Página individual para cada post
  - Sistema de comentários
  - Página de categorias
  - Página de contato
- 🔍 **Barra de busca** - Encontre posts facilmente
- 🏷️ **Sistema de tags** - Organização por temas
- 📊 **Posts recentes** - Destaque para conteúdo novo
- 🔗 **Redes sociais** - Compartilhamento integrado

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Estilização moderna com Flexbox e Grid
- **GitHub Pages** - Hospedagem gratuita (opcional)
